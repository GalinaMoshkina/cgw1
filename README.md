S - обозначение сигнализации.  
При изменении r и λ также должны поменяться и P(S|раб), P(S|не раб), и P(m≥2 ударов).  

## Что нужно найти:  
- Какова вероятность того, что вся система работает, если мы не знаем, сколько ударных событий произошло?  
- Какова полная вероятность срабатывания сигнализации?  
- Какова вероятность работы системы при срабатывании сигнализации?  

Нет некоторых значений: **P(S|раб), P(S|не раб), и P(m≥2 ударов)**, их найдем через код, который прикреплен в данном репозитории.  
  
**Формула Пуассона**  
P(m) = (λ^m × e^{-λ}) / m!  
**Для системы вероятность работы вычисляется по биномиальному распределению**  
P(раб|m) = Σ[C(i,6) × r_new^i × (1-r_new)^{6-i}] для i=4,5,6  
**Сигнализация срабатывает с вероятностью, зависящей от количества неисправных модулей**  
P(срабатывания) = min(0.95, 0.05 + 0.15 × количество неисправных)  
P(S|раб) = 0.111  
P(S|не раб) = 0.509  
P(m≥2 ударов) = 0.888  
Система устойчива: при двух и более ударах сохраняется работоспособность с вероятностью 88.8%, тем временем сигнализация нуждается в оптимизации, тк обнаруживает около половины реальных отказов.
